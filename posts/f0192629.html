之前在写硕士毕业论文的时候使用chatgpt的频率还蛮高的，但是有时候会出现

【Please try again later. If you are using a VPN, try turning it off. Check the status page for information on outages. 】报错

有时候也会出现其他类似的登录问题，比如：

  1. 打开 ChatGPT 提示 「Unable to load site」 的错误
  2. 网页提示 「Oops! We ran into an issue while authenticating you.」 的错误
  3. 使用提示 「Something went wrong」 的错误
  4. 登录时，提示拥堵 「Oops! We ran into an issue while signing you in」
  5. 使用时，左侧的聊天记录加载不出来，「Unable to load history」 错误

归根到底这其实都是网络问题，原因是当前用的科学上网的节点或服务不行，比如国内和hk的节点就不行，需要换到其他地区（尽量使用非亚洲地区）的节点试试。

后来接触到一个不错的网站 wildcard，wildcard是一个虚拟海外卡平台，通过银联购汇，支持支付宝，注册后会得到一个账户，可用于升级ChatGPT Claude，订阅YouTube premium，开通美区Apple ID等

【从这个链接 wildcard 进去可以获得至少8.7折优惠】

虽然现在chatgpt不需要注册即可使用，但没有账号就无法保存历史聊天记录，更不能使用升级版的服务，因此建议还是注册一个账号比较好，比如可以使用proton邮箱注册，

也可以通过card的「海外邮箱」获得gmail谷歌邮箱，

问题迎刃而解，并且按照 wildcard 提供的教程顺利升级了chatgpt4.0，详细的升级图文教程还可以见顶部⬆️的表格

以后读博的话，不管是写论文还是做科研应该都会长期使用chatgpt4.0，人类的本质就是复读机嘛，如果博士课题也能交给chatgpt去做出来该多好啊…放在以前这就是痴人说梦，但按照现在AI发展的速度来看，我们距离那一天越来越近了。

就像OpenAI 首席执行官 Sam Altman 说的那样，“十年前，传统观点认为人工智能首先会影响体力劳动，然后是认知劳动，也许有一天它可以从事创造性工作。现在看来，事情会按照相反的顺序进行。”而他说这话的时候是2022年DALL•E 2即将推出的前夕，但你看现在，DALL•E 3 都出来了，所以就如同他所预测的一样，人类智力的发展速度已经远远跟不上AI的迭代速度了，AI首先挑战的是从事创造性工作的人们，但它也即将改变整个世界。

希望可以浅浅地跟上AI发展的脚步，争取不会被时代淘汰。
