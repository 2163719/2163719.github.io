Claude 3 打响科技“军备”竞赛之二——【如何升级 Claude 3？】

什么是 Claude 3？

2024年3月4日，Anthropic 宣布推出 Claude 3 模型系列，它为广泛的认知任务树立了新的行业基准。该系列包括三种最先进的模型（按功能升序排列）：Claude 3 Haiku、Claude 3 Sonnet 和 Claude 3 Opus。每个后续的模型都提供越来越强大的性能，允许用户为其特定应用选择智能、速度和成本的最佳平衡。

Opus 和 Sonnet 现已可在 claude.ai 中使用，而 Claude API 现已在159 个国家/地区广泛使用。Haiku 即将推出。

可以说Claude 3 的发布正式打响了各大科技公司的“军备竞赛”号角📯📯📯

如何注册Claude账号

升级之前首先需要注册Claude账户，

⚠️ ：然而 Claude 封号非常激进，相信很多人在注册Claude 3 的时候都踩了很多坑，我就试过proton邮箱，如下图所示，几分钟之后被封掉

但封号跟支付方式并没有直接的关系，Claude 一旦监测到你用 科学上网的工具（tizi），就会直接封你的账户。

因此，通过自己亲身经历和实验，我终于找到了解决办法，注册成功Claude的三个关键要素是：

  • Google邮箱账号
  • 海外手机号
  • 海外网络

这三个要素必须同时满足，在CN地区被屏的情况下确实是很困难的事情，这个时候就得推荐一下 wildcard 了，它是一个虚拟信用卡平台，通过银联购汇，支持支付宝，注册后会得到一个账户，在这里可以用于升级订阅Claude 3，也完全可以用于升级ChatGPT Plus等服务，具体支持的服务如下所示，可以说开一张卡，用处多多啦~~

开完card之后，【网络问题】可以借助⬇️解决

按照指引一步步注册Claude页面进行到手机验证码阶段，

然后申请一个手机号，

选择要注册的服务，

你会得到一个手机号，然后发送验证码，将验证码填入，点击创建账户

升级Claude 3

创建完账户之后就可以升级了

登录进入 Claude 页面，点击下面的【Upgrade to Claude Pro】

点击【subscribe Now】

填写card上面的信息即可，

至此就升级完成啦~
